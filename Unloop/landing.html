Washington Re-Entry Connector

Resources for formerly incarcerated people

The WA Re-entry Connector is meant to help people in the state of Washington re-enter their communities following incarceration. Please complete the following three statements and then click "Show results now"

I'm looking for services in this county or these counties:

You can select multiple counties if applicable.

I'm looking for these services:

You can select multiple services

I identify as:

You can only select one


We ask for your county so that you are only shown services which match your release county.

This allows you to filter your results for the services you are looking for now.

We ask because some services are provided based on gender.
If you select:

  • “Female”, you’ll see services that are female-only or non gender-specific.
  • “Male”, you’ll see services that are male-only or non gender-specific.
  • “Prefer not to specify”, you’ll see all services.
